# Implementation Plan: Add Game Instructions & Enhance Play Button

## Overview
Add game instructions modal and make the Play Pet Game button more prominent/attractive.

## Tasks

### 1. Create Game Instructions Modal Component
**File**: `components/GameInstructionsModal.tsx` (NEW)

**Features**:
- Cyberpunk-styled modal with glassmorphism, neon borders, and glow effects
- Clear sections: Objective, How to Play, Scoring, Tips
- Responsive design (works on mobile/desktop)
- "Don't show again" checkbox with localStorage persistence
- Close button with hover effects
- Keyboard accessible (ESC to close)

**Content**:
- **Objective**: Catch the virtual pet by aligning it with the red target inside the PiP window
- **How to Play**: 
  1. Click "Play Pet Game" to open Picture-in-Picture window
  2. Move the PiP window around your screen
  3. Position the pet inside the red target circle at the center
  4. Score +1 when pet is caught
  5. Pet teleports to new location after 1 second
- **Scoring**: Each catch = +1 point, score persists across sessions
- **Tips**: Watch for teleport burst effect, use X/Y controls to pin position
