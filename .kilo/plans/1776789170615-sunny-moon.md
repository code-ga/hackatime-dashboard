# Plan: User Profile Page - View Other Users by Username

## Goal
Create a page at `/user/[username]` that displays a user's public Hackatime profile with:
- User info card with avatar, username, trust factor
- Projects list with stats (languages, coding time, heartbeats)
- Activity timeline from heartbeat spans

## Design Style: Cyberpunk/Terminal
- Dark background (#0a0a0f)
- Neon accent colors (cyan #00ffff, magenta #ff00ff, green #00ff00)
- Glowing effects on cards and buttons
- Monospace fonts (JetBrains Mono)
- Grid/circuit-like decorative elements

## API Endpoints to Use
1. `GET /api/v1/users/{username}/projects` - Get user projects list
2. `GET /api/v1/users/{username}/projects/details` - Get detailed project stats
3. `GET /api/v1/users/{username}/heartbeats/spans` - Get activity timeline

## Implementation Plan

### Phase 1: Setup shadcn components
```bash
npx shadcn@latest add input skeleton badge progress tabs avatar
```

### Phase 2: Create User Profile Page
- `app/user/[username]/page.tsx` - Main dynamic route page

### Phase 3: Create Components
- `components/user-search.tsx` - Username search form
- `components/user-profile-card.tsx` - Animated profile header
- `components/project-list.tsx` - Project cards with stats
- `components/activity-timeline.tsx` - Visual activity timeline

### Phase 4: Styling & Animations
- Cyberpunk theme with neon glow effects
- Staggered animations using tw-animate-css
- Loading skeletons
- Error/empty states

## Page Structure
```
/user/[username]
├── Search form (at top or via URL)
├── UserProfileCard
│   ├── Avatar (glow effect)
│   ├── Username
│   ├── Trust badge
│   └── Quick stats
├── ProjectList (Tabs: Recent | All)
│   └── ProjectCard[] (staggered animation)
└── ActivityTimeline (visual spans)
```

## Success Criteria
1. Accessible via /user/[username] URL
2. Shows user info, projects with stats, activity timeline
3. Cyberpunk visual style with neon glow effects
4. Smooth animations throughout
5. Loading and error states handled
