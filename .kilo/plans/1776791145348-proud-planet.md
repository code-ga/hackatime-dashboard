# Activity Timeline Snake Animation Plan

## Overview
Add a GitHub-style snake animation to the activity heatmap where a visual "snake" moves through each date cell sequentially, highlighting them as it "eats" the activity data from the first day to the last day of the year.

## Current State
- Activity heatmap displays last 30 days of coding activity
- Each cell shows intensity based on coding duration
- Uses colors: muted (no activity), magenta (low), cyan (medium), green (high)

## Requirements
1. **Full Year Display**: Currently shows only 30 days - need to expand to full year (365 days)
2. **Snake Animation**: Visual snake head moves cell by cell from start to end
3. **Cell Highlighting**: Each cell lights up as snake passes
4. **Trail Effect**: Previous cells retain a subtle glow/highlight after being "eaten"
5. **Smooth Movement**: Eased animation between cells for natural movement
6. **Play/Pause Control**: Allow replay of the animation

## Technical Approach

### 1. Expand Heatmap Data Range
- Change from last 30 days to full year (365 days or last 365 days with data)
- Generate all dates in range, filling missing dates with 0 duration
- Sort dates chronologically (oldest to newest)

### 2. Animation Architecture
Use CSS animations with staggered delays:

```css
/* Snake head highlight */
@keyframes snake-head {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.3); }
}

/* Cell "eaten" glow effect */
@keyframes cell-eaten {
  0% { box-shadow: none; }
  100% { box-shadow: 0 0 6px var(--current-color); }
}

/* Staggered animation delays per cell */
.cell-animated {
  animation: cell-eaten 0.3s ease-out forwards;
  animation-delay: calc(var(--cell-index) * 100ms);
}
```

### 3. Implementation Steps

#### Step 1: Modify Data Generation
- `getActivityHeatmap()` → Generate dates for full year
- Fill missing dates with 0 duration
- Return chronologically sorted array

#### Step 2: Add Animation State
- Track `isAnimating` and `animationProgress`
- Use `requestAnimationFrame` or CSS delays for timing
- Snake position = `Math.floor(animationProgress * totalCells)`

#### Step 3: Create Snake Visual Component
- SVG snake head icon OR CSS-drawn head
- Position over current cell
- Animated pulse effect on head

#### Step 4: Update Cell Rendering
- Add `data-cell-index` attribute for animation timing
- Apply staggered `animation-delay`
- Conditional glow based on animation progress

#### Step 5: Add Controls
- Play/Replay button
- Optional: Speed control (1x, 2x, 0.5x)

### 4. Key Files to Modify
- `components/user/activity-timeline.tsx` - Main component logic
- `app/globals.css` - Add animation keyframes

### 5. Animation Details
| Property | Value |
|----------|-------|
| Duration per cell | 100ms |
| Total animation | 365 * 100ms = 36.5s |
| Snake head size | 1.3x normal cell |
| Trail glow | 0.3 opacity added to base color |
| Easing | `ease-out` for natural deceleration feel |

### 6. Performance Considerations
- Use CSS `will-change: transform` on animating elements
- Batch DOM updates
- Consider reducing to 52 weeks (weekly columns) for mobile
