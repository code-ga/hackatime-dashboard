# Plan: Add Game Instructions & Enhance Play Button Visibility

## Problem
1. Users don't know how to play the pet game (no instructions)
2. The "Open Pet" button is visually inconspicuous (small gray text in top-right corner)

## Solution Overview

### 1. Create an Instructions Modal Component
- New component: `components/GameInstructionsModal.tsx`
- Cyberpunk-styled modal with neon borders and glow effects
- Shows game objective, controls, scoring mechanics, and tips
- Triggered by a "How to Play" button near the main game button

### 2. Redesign the Pet Game Launcher Button
Replace the current small text link with an eye-catching button in the Dashboard's HeroSection.

**Button Features:**
- **Location**: Next to the trust factor badge in `HeroSection` (line 220-221)
- **Style**: Cyberpunk-primary button using `Button` component with custom variant
- **Visual Elements**:
  - Neon glow (cyan or magenta)
  - Pulsing animation to attract attention
  - Pet icon (simple SVG inline)
  - "Play Pet Game" text with glow effect
  - Larger size (`size="lg"` or custom)
- **Interactions**:
  - Hover: Scale up, intensify glow
  - Click: Opens PiP window and instructions modal (if first time)

### 3. Add First-Time User Guidance
- Use `localStorage` flag to detect first-time players
- Show instructions modal automatically on first click
- Add a "Don't show again" checkbox in the modal

### 4. Update PetPiP Component
- Add a small "? Instructions" button in the PiP controls panel
- Clean up current button styling to match cyberpunk theme
