# Dashboard Redesign Plan: Cyberpunk/Hacker Style

## Overview
Redesign `/dashboard` page with a cyberpunk/hacker aesthetic using all available Hackatime API data.

## Current State
- Basic card-based layout with generic shadcn/ui styling
- Displays: user info, hours, streak, current activity, top projects
- Missing: full use of available data fields, modern cyberpunk visuals

## Available Data (Currently Underutilized)

### User Data
- `id`, `emails`, `github_username`, `trust_factor` (trust_level, trust_value)

### Hours Data
- `start_date`, `end_date`, `total_seconds`

### Streak Data
- `streak_days`

### Latest Heartbeat (FULL DATA)
- `id`, `created_at`, `time`, `category`, `project`, `language`, `editor`, `operating_system`, `machine`, `entity`

### Projects Data (FULL)
- `name`, `total_seconds`, `percent`, `most_recent_heartbeat`, `languages`, `archived`

## Implementation Plan

### 1. Visual Design System
- **Background**: Dark theme with `.cyberpunk-grid` pattern
- **Color Palette**: Neon cyan (#00ffff), magenta (#ff00ff), green (#00ff00)
- **Effects**: Glow effects (`neon-cyan`, `glow-cyan`), gradient overlays
- **Typography**: Monospace fonts (JetBrains Mono)
- **Cards**: Glassmorphism with backdrop blur, border glow, animated gradients

### 2. Page Layout

### 3. New Components & Features

#### Hero Section
- Animated avatar with glow border matching trust level
- Username with neon glow effect
- Trust level badge (color-coded: high=green, medium=cyan, low=magenta)
- Quick stats: total hours, daily average, current streak, trust value
- Animated stat counters

#### Hours Card (Enhanced)
- Total coding time with large neon display
- Date range info
- Visual time bar/indicator

#### Streak Card (Enhanced)
- Current streak with fire/flame animation effect
- Display styling similar to GitHub contributions

#### Current Activity Card (Enhanced - USE ALL DATA)
- Project name (with link styling)
- Language badge
- Editor
- Operating system (NEW)
- Machine name (NEW)
- Entity (NEW)
- Category (NEW)
- Created at timestamp (formatted)

#### Projects Grid (Enhanced)
- Card-based layout for each project
- Show: name, time, percent bar, languages used, archived status
- Color-coded progress bars
- Hover effects with glow

#### System Info Panel (NEW)
- Display editor, OS, machine name, entity
- Terminal-style display

### 4. Animations
- Fade-in on mount with staggered delays
- Pulse animations on live data
- Stat counter animations
- Glow pulse effects
- Card hover transitions

### 5. Loading/Error States
- Cyberpunk-themed skeleton loaders
- Glitch effect on error
- Terminal-style error messages

## Files to Modify
- `app/dashboard/page.tsx` - Main dashboard

## Design Patterns to Follow
- Use existing `.neon-*` and `.glow-*` utility classes from globals.css
- Follow `user-profile-card.tsx` styling patterns
- Use shadcn/ui components with custom styling
- Implement `tw-animate-css` animations

## Style Focus: VISUAL IMPACT
- Large neon glow effects on all major elements
- Glassmorphism cards with backdrop blur
- Animated stat counters with number transitions
- Fire/streak animations for streak display
- Hover effects with glow intensification
- Gradient overlays and scan-line effects
- Priority: Visual hierarchy and aesthetic appeal over maximum data density

## Completion Criteria
1. Page displays all user data available from API
2. Visual style matches cyberpunk aesthetic (neon, glow, dark theme)
3. Responsive layout (mobile-first)
4. Smooth animations and transitions
5. Proper loading and error states
6. High visual impact with dramatic neon effects
